# Welcome to AOTE PMS Storybook

This Storybook is for local development only and is not deployed to Vercel.

## Component Categories

- **Dialogs**: Modal components like PaymentModal, RateModal, VoucherModal
- **Forms**: InlineEdit, Input patterns
- **Tables**: Session tables, Payment tables with sorting
- **Tabs**: OverviewTab, BillingTab, PersonalTab patterns

## Usage

Run Storybook locally with:

```bash
npm run storybook
```

Build static Storybook with:

```bash
npm run build-storybook
```

## Note

This Storybook uses Ant Design components. All components have been migrated from Material UI to Ant Design.
